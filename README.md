# Mariachi San Isidro

Sitio web oficial para un grupo de mariachi orientado a presentar su propuesta musical, mostrar a sus integrantes, compartir su repertorio y facilitar el contacto para cotizaciones por WhatsApp.

Este proyecto nace como una experiencia web moderna para comunicar la identidad del grupo de forma elegante, cercana y profesional. La idea es ofrecer una landing page clara y visual, con información útil para clientes que buscan música para serenatas, bodas, cumpleaños, eventos corporativos y celebraciones especiales.

## Estado del proyecto

El sitio se encuentra en desarrollo y actualmente se trabaja en:

- la mejora visual de la experiencia de usuario,
- la organización del repertorio musical,
- la presentación de integrantes y galería,
- el flujo de cotización y contacto por WhatsApp,
- la optimización responsive para distintos dispositivos,
- y la consolidación del contenido en español e inglés.

## Tecnologías utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- Radix UI
- Material UI
- Framer Motion / Motion
- Lucide React

## Funcionalidades principales

- Sección de inicio con presentación del grupo.
- Información sobre la historia y valores del mariachi.
- Listado de integrantes con sus instrumentos.
- Repertorio filtrable por ocasión, género y ambiente.
- Galería de imágenes y videos.
- Formulario de cotización con envío directo a WhatsApp.
- Diseño responsivo y bilingüe.

## Instalación y desarrollo

### Requisitos

- Node.js 18 o superior
- npm o pnpm

### Instalar dependencias

```bash
npm install
```

### Ejecutar en modo desarrollo

```bash
npm run dev
```

### Generar build de producción

```bash
npm run build
```

## Estructura general

- `src/app`: lógica principal de la aplicación.
- `src/app/components`: componentes reutilizables y secciones del sitio.
- `src/app/data`: contenido estático del proyecto.
- `src/styles`: estilos globales, tipografías y tema visual.

## Nota

El proyecto toma como base un diseño de referencia de Figma y se ha adaptado para funcionar como una web funcional, presentable y fácil de mantener.
